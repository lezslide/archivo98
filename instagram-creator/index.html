Instagram Creator App

Tu mini ManyChat propio, pero sin ManyChat.

Esta base escucha comentarios con webhook y dispara respuestas privadas usando la API oficial de Instagram para cuentas profesionales tipo creador.

Endpoints incluidos

  • GET /api/instagram-config-test prueba token y account ID.
  • GET /api/instagram-webhook verifica el webhook con Meta.
  • POST /api/instagram-webhook procesa eventos de comentarios.
  • POST /api/instagram-private-reply manda una respuesta privada manual.

Variables de entorno

INSTAGRAM_ACCESS_TOKEN=
INSTAGRAM_ACCOUNT_ID=
INSTAGRAM_VERIFY_TOKEN=
INSTAGRAM_APP_SECRET=
INSTAGRAM_GRAPH_VERSION=v24.0
INSTAGRAM_KEYWORD_RULES=[{"keyword":"precio","reply":"Te escribo por privado ahora."}]

Alta en Meta

  1. Crea una app en Meta for Developers.
  2. Activa Instagram API con Instagram Login.
  3. Genera token para tu cuenta de creador.
  4. Configura el webhook apuntando a /api/instagram-webhook.
  5. Suscribe el campo comments.

Chequeo rápido

Presiona el botón cuando ya hayas cargado las variables de entorno.
