# Filo Construcciones Descargada

Esta carpeta contiene una copia local del build publicado en:

https://filo-construcciones.vercel.app

Incluye el `index.html`, el bundle JavaScript y el CSS que expone la app publica.

## Uso

```powershell
npm run dev
```

Luego abrir la URL local que muestre `serve`.

## Nota

Esto no es el codigo fuente original de React/Vite. Desde una URL publica solo se puede recuperar el build compilado y los recursos publicos expuestos por el navegador. Para tener componentes, rutas, configuracion original y estructura editable completa hace falta el repositorio fuente o el proyecto de Vercel.
